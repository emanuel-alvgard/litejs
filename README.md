# lite.js
------
*Data oriented DOM wrapper and UI library.*
------
**Examples**
````
// CREATE
element = create_element("div");

// GET
get_x(element)
get_y(element)

// SET
set_x(element, root, 100.0);
set_y(element, root, 100.0);

// DELETE
delete_element(element);

````
